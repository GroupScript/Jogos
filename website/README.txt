1- Estrutura

A pasta website contém três pastas e os arquivos .html: 
Na primeira pasta nomeada de css, fica localizado todos os arquivos .css, inclusive de plugins de jQuery usados no site e o style.css criado pelo grupo;
Na segunda pasta nomeada de imagens, ficam localizada todas as imagens, imagem de fundo do site, imagens dos plugins etc;
Na terceira pasta nomeada de js, fica localizado todos os arquivos .js, de plugins e o script.js criado pelo grupo;
A pasta website contém quatro arquivos .html.


2- Funcionalidade

É um simples site de jogos, contendo imagens de alguns dos principais jogos antigos, login na página principal, uma página de cadastro e outra de contato.


3- Configuração

Para acessa o site basta abrir o arquivo index.html, localizada na pasta website.
